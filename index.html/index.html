Introduction

This page is an experiment to explore simple steps web developers can take to make their projects easier to see and more user-friendly. Not every user is familiar with existing tools for visual aid or the various hidden keystrokes pre-installed on their operating system. This page is simply an exercise in implementing basic button-placement tools for the visually impaired and/or those who are less familiar with computers. These methods do not need to be implemented everywhere, but they could be especially useful for text-heavy web pages.
